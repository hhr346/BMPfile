## 运行

运行main.py即可，读取步骤在`ReadBMPFile.py`里，具体对lena的分量进行处理后输出的步骤在`ModifyBMPFile.py`里可以进行更改。

> 值得注意的是，bmp的内部存储格式也很多，参考链接里的读取方式并不适用于所有类型。
